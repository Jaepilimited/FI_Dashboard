# FI Monthly Report — Design Spec
**Date:** 2026-06-10  
**Status:** Approved

---

## Overview

월간 재무 성과를 경영진/임원이 빠르게 파악할 수 있는 스코어카드형 HTML 리포트.  
기존 FI Dashboard의 Flask + BigQuery 인프라를 그대로 활용하며, 독립 HTML 파일로 먼저 구현한 뒤 대시보드 통합 여부는 사용자가 결정한다.

---

## Decisions Made

| 항목 | 결정 |
|------|------|
| 형식 | 독립 HTML 파일 (Flask route `/report` 로 서빙) |
| 대상 | 경영진/임원 |
| 비교 기준 | 전월 대비 (MoM) |
| 분석 차원 | 조직 / 판매유형 / 제품라인 / 국가·대륙 전부 |
| 스타일 | 클린 화이트 (흰 배경, 검정 테두리 헤더) |
| 레이아웃 | KPI 전폭 + 2×2 분석 그리드 |

---

## Architecture

### 데이터 흐름

```
브라우저 → GET /report?month=2026-05
         → Flask: render report.html (템플릿)
         → 페이지 로드 후 JS가 기존 API 호출:
             /api/kpi?months=2026-05
             /api/kpi?months=2026-04          ← 전월 자동 계산
             /api/department?months=2026-05,2026-04
             /api/sales-type?months=...
             /api/line?months=...
             /api/country?months=...
         → 수신한 JSON으로 테이블/수치 렌더링
```

### 전월 자동 계산 규칙
- 선택 월이 `YYYY-MM` 형식이면 JS에서 `-1month` 계산
- 예: `2026-05` → 전월 `2026-04`
- `2026-01` → 전월 `2025-12` (연도 넘김 처리 포함)

### 신규 파일
- `templates/report.html` — 리포트 템플릿 (단일 파일)
- `app.py` — `/report` 라우트 추가 (login_required 데코레이터)

### 기존 API 재사용 (변경 없음)
- `/api/kpi` — KPI 5개 계산
- `/api/department` — 조직별 집계
- `/api/sales-type` — B2B/B2C 집계
- `/api/line` — 제품라인 집계
- `/api/category` — 카테고리 집계
- `/api/country` — 국가별 집계

## Page Structure

### 1. 헤더
- 왼쪽: "월간 재무 성과 리포트" 제목 + "Financial Intelligence Monthly Scorecard" 부제
- 오른쪽: 월 선택 `<select>` (BigQuery에 있는 월 목록 자동 로드) + 생성일
- 헤더 하단: 검정 2px 구분선

### 2. KPI 스코어카드 (전폭)
5개 카드, 가로 균등 배치:

| 카드 | 표시 내용 | 증감 색상 규칙 |
|------|-----------|---------------|
| 매출 | 당월 금액 + 전월比% | 상승=초록, 하락=빨강 |
| 매출원가 | 당월 금액 + 전월比% | 상승=빨강(비용↑), 하락=초록 |
| 매출총이익률 | 당월 % + 전월比 pp | 상승=초록, 하락=빨강 |
| 판관비 | 당월 금액 + 전월比% | 상승=빨강(비용↑), 하락=초록 |
| 영업이익률 | 당월 % + 전월比 pp | 상승=초록, 하락=빨강 |

- 카드 상단 border: 이익성 지표(총이익률·영업이익률)는 전월比에 따라 동적으로 초록/빨강 2px 라인
- 각 카드 하단: 전월 수치 소자로 표시

### 3. 2×2 분석 그리드

#### [1] 조직별 (Department)
- 컬럼: 순위 뱃지 | 조직명 | 매출 | 전월比% | 영업이익률
- 매출 기준 내림차순 정렬
- 순위 뱃지: 1위=검정, 2위=회색 진, 3위=회색, 이하=연회색

#### [2] 판매유형 (B2B / B2C)
- 컬럼: 유형 | 매출 | 비중(미니 바) | 전월比% | 영업이익률
- 비중 미니 바: 가로 바 + % 숫자 표시
- 하단 구분선 아래: 카테고리별 상세 (Category API)

#### [3] 제품라인 (Line)
- 컬럼: 순위 | 라인명 | 매출 | 비중(미니 바) | 전월比% | 매출총이익률
- 전체 매출 대비 비중 바 표시

#### [4] 국가별 (Country)
- 컬럼: 순위 | 국가명 | 대륙 | 매출 | 전월比% | 영업이익률
- Top 10 표시 (기존 API LIMIT 30 중 상위 10개)

### 4. 푸터
- 왼쪽: "단위: 백만원 / 기준: BigQuery FI 데이터" 주석
- 오른쪽: "FI DASHBOARD" 로고 텍스트

## UX Details

### 월 변경 인터랙션
1. `<select>` 변경 시 JS가 URL querystring 업데이트 (`?month=YYYY-MM`)
2. 모든 API 재호출 (현재 월 + 전월 동시)
3. 로딩 중: 각 섹션에 스켈레톤 shimmer 효과
4. 에러 시: 해당 섹션에 "데이터를 불러올 수 없습니다" 메시지

### 숫자 포맷
- 금액 ≥ 1,000M: `X,XXX M` 또는 `X.X B` (10억 이상은 B 단위)
- 금액 < 1,000M: `XXX M`
- 비율: 소수점 1자리 (예: 21.9%)
- pp 변화: ▲/▼ 기호 + 소수점 1자리 + "pp"

### 접근 제어
- `@login_required` 데코레이터 적용 (기존 세션 기반 인증 재사용)
- 비로그인 시 `/login` 리다이렉트

## Out of Scope
- YoY 비교 (전년 동월 대비)
- PDF 내보내기
- 이메일 자동 발송
- 대시보드 네비게이션 통합 (사용자 결정 후 별도 작업)
- 차트/그래프 (스코어카드 방식 — 테이블+미니바로 대체)
